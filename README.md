# AB12 (Test)

This is an API project built with dotnet 6 to be used to test developers.

## Environment

- .Net framework version: 6.0
- Database: MySql 8

## Requirements

### Build the following APIs

- Create Order.
- Get All Orders.
- Get Order By Id.
- Update Order.
- Delete Order.

### Bonus Challenge

- Use AutoMapper.
- Implement Pagination for List Results.
- Add Image for products.
